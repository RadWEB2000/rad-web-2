Aplikacja webowa jest zbudowana o wyżej wymieniony framework NextJS, jednakże nie są to wszystkie technologie użyte w tym projekcie. Użyłem styled-components, technologii, która umożliwia nadawanie wyglądów CSS z wykorzystaniem JavaScript oraz TypeScript do kontrolowania wprowadzanych danych, poprawności przekazywanych komponentów, a także właściwości (***properties** — czyli żargonowych propsów*)
